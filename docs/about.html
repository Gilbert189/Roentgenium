A bot for the TBGs



Nihonium is a bot made for the TBG forums, designed to be able to parse input and give responses.


...that's might not sound impressive, until you realize that the TBGs don't have any sort of API; the bot uses raw HTTP requests to recieve input and send output.


It's currently in early development, meaning that things can change at any time, so you shouldn't expect something to work the same way it does currently after a month, a few weeks, or even a few days.



If you'd like to know more about what commands it supports, check out the Commands page.


If you'd like to know more about the internal functions, check out the Developer page.
